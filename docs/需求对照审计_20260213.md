## 四、尚未实现 / 需补充的需求

### P1 — 需求明确但实现不完整

1. **`date_utils.py` 缺少 A 股节假日处理** — 当前只排除周末，春节/国庆等长假期间会误判为交易日，影响数据采集和榜单生成的准确性

### P2 — 需求标注"预留"但尚未真正实现

2. **`tracking/auto_updater.py`** — 自动更新调度器，需要定时触发采集+清洗+研究的完整流程
3. **`tracking/portfolio_reviewer.py`** — 组合回顾报告，需要 AI 生成周/月度持仓回顾
4. **`ingestion/iwencai_source.py`** — 问财条件选股数据源
5. **`ingestion/audio_transcriber.py`** — 音频转写（电话会/路演）

### P3 — FastAPI 前端缺少独立页面

6. **"数据管理"在 FastAPI 版本中没有独立页面**，功能被拆分到 `/data`（采集清洗）和 `/settings/structured`（结构化数据）。Streamlit 版有独立的 `data_admin.py`

## 五、超出需求的新增实现

- 市场总览页面 (market.html) — 指数行情、行业资金流、北向资金、融资融券

- 个股/热点跟踪研究 (followup) 页面
